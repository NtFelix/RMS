# Product Overview

## Rent-Managing-System (RMS)

RMS is a German property management web application designed for landlords and property managers to handle:

- **Mieter** (Tenants) - tenant information, move-in/out dates, contact details
- **Wohnungen** (Apartments) - apartment listings with size, rent, and house associations  
- **HÃ¤user** (Houses) - property buildings that contain apartments
- **Finanzen** (Finances) - income/expense tracking per apartment
- **Betriebskosten** (Operating Costs) - utility costs and expense calculations
- **Aufgaben** (Tasks) - todo management for property maintenance

## Key Features

- Multi-tenant SaaS with Stripe subscription management
- Data export capabilities (PDF, CSV)
- Responsive dashboard with charts and analytics
- User authentication via Supabase Auth
- German language interface with localized terminology

## Business Model

Subscription-based with different tiers limiting the number of apartments (Wohnungen) a user can manage. Trial periods and subscription management handled through Stripe integration.
